## Vue Element项目

### 初始化项目
1. 安装 vue-cli脚手架
2. 通过vue脚手架创建项目
3. 配置vue路由
4. 配置 Element-UI组件库
5. 配置axios库
6. 初始化git远程仓库
7. 将本地项目托管到github或码云



### node后台
1. 创建数据库
2. node app.js 运行起来没有报错就表示后台可以运行了


### 通过脚手架创建项目

```shell

vue init webpack my-project

# 安装依赖

npm install less-loader
npm intsall less
npm install element-ui


```
